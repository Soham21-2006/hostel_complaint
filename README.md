# 🏠 Hostel Complaint Management System

A complete full-stack web application for managing hostel complaints efficiently. Students can submit complaints, track their status, and admins can manage and resolve complaints with real-time updates.

## ✨ Features

### For Students
- 🔐 **Secure Registration & Login** - Create account and login securely
- 📝 **Submit Complaints** - Report issues like Electrical, Plumbing, Cleaning, etc.
- 👀 **Track Status** - Monitor complaint progress (Pending → In Progress → Resolved)
- 👤 **Profile Management** - View personal details and room information
- 📱 **Mobile Responsive** - Works perfectly on all devices

### For Admin
- 📊 **Dashboard Analytics** - View statistics (total complaints, pending, resolved)
- 📋 **Manage All Complaints** - View all student complaints in one place
- ✅ **Update Status** - Change complaint status and add remarks
- 👨‍🎓 **Student Overview** - See which student reported which issue
- 🔔 **Real-time Updates** - Students see status changes instantly

## 🛠️ Technology Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB Atlas (Cloud) |
| **Authentication** | JWT (JSON Web Tokens) |
| **Styling** | CSS3 with Flexbox/Grid, Font Awesome Icons |
| **Deployment** | Render / Cyclic (Free Hosting) |
